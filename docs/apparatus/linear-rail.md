# Linear Rail System

The linear rail provides 1000mm of approach distance for the servicer satellite to travel toward the defunct satellite on the gimbal.

---

## Rails

| | |
|---|---|
| **Type** | HGR15 linear guide rails |
| **Length** | 1000mm |
| **Quantity** | 2 rails, 4 HGH15CA carriage bearings |
| **Link** | [Amazon.ca](https://www.amazon.ca/ANWOK-HGR15-1000mm-HGH15CA-Carriage-Bearing/dp/B09VC59SY1) |

The servicer satellite sits on a thrust bearing mounted to the rail carriages. This allows the servicer to rotate freely on the rail while being driven linearly.

---

## Belt Drive

The approach motion along the rails is driven by a GT2 timing belt system. The belt motor is housed in the base of the gimbal.

| | |
|---|---|
| **Belt** | 5M GT2 timing belt |
| **Kit includes** | Belt, pulleys, tensioners, attach blocks |
| **Link** | [Amazon.ca](https://www.amazon.ca/dp/B08SMFM3Z6) |

---

## How It Works

1. Servicer satellite is mounted on rail carriages via a thrust bearing
2. Belt connects to the carriage assembly
3. Belt motor (stepper, in gimbal base) drives the belt
4. Servicer translates linearly toward the gimbal/defunct satellite
5. 1000mm travel provides enough approach distance for the arm to reach and capture
