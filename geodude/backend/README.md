# GEO-DUDe Backend

This folder runs on the GEO-DUDe Pi.

Responsibilities:

- STM32/Nucleo serial bridge
- I2C sensors such as ADS1115 and IMU
- PCA9685/gimbal support where still needed
- local hardware API on port `5000`

For now, the GEO-DUDe Pi runs the main checkout only. Worktrees are only planned for groundstation frontends.
